Welcome to boundfit's documentation!
====================================

BoundFit determines upper and lower boundaries to a given data set using the
generalised least-squares method described in Data boundary fitting using a
generalised least-squares method (Cardiel 2009, MNRAS, 396, 680; hereafter
C09).
